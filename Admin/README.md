# Grey_InfoTech Admin Backend

This folder now runs as a TypeScript Express/EJS backend.

## Scripts

```bash
npm start
npm run dev
```

## Notes

- `npm start` runs `app.ts` through `tsx`
- `npm run dev` watches `app.ts` for changes
- Authentication routes are handled in `routes/auth.ts`
- Protected dashboard routes are handled in `routes/admin.ts`
